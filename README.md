# Pivot Real Estate Development

A modern, high-performance web application for Pivot Real Estate Development, a leader in luxury, eco-friendly property development.

## Project Overview

Pivot Real Estate Development specializes in acquiring, building, and delivering luxurious, eco-friendly living spaces that combine elegance, innovation, and environmental responsibility. This project serves as the digital showcase for their properties, services, and commitment to excellence.

## Key Features

- **Property Showcase**: Detailed views of luxury residential and commercial properties.
- **Service Offerings**: Information on acquisition, construction, development, and rentals.
- **Eco-Friendly Focus**: Highlighting sustainable building practices and environmental responsibility.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices.
- **Interactive UI**: Smooth animations and transitions using Tailwind CSS and shadcn/ui.

## Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Project Structure

- `src/app`: Next.js App Router pages and layouts.
- `src/components`: Reusable UI components and page sections.
- `src/lib`: Utility functions and data.
- `public/assets`: Static images and assets for the project.
